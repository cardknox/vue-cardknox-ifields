# vue-cardknox-ifields-test

## Project setup
1. 
```
npm install
```

2. make sure the component is built
```
cd ../vue-cardknox-ifields
npm install
npm run build
```

3. Add your ifields key to `src/App.vue`
4. Add your gateway key to src/components/submit.vue
